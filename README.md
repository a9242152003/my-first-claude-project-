# 放量突破交易策略(BTC / 加密 / 高波動動能標的)

一套給 **TradingView Pine v5** 的趨勢交易策略,核心是「**放量突破進場 + 風險定額倉位 + 牛熊濾網 + ATR 移動停利**」。
從「全倉 20x 會爆倉」一路演化到「風險定額、槓桿自動浮動、跨 16 市場零爆倉」的可落地版本。

---

## 🚀 直接用哪支?

> **`breakout_v5_main_bybit.pine` ← 主策略,用這支。**

- 價格用本圖(= 你下單的交易所,如 Bybit 永續),放量判定借幣安現貨量。
- 進場標註好(進場價 / 倉位$ / 本金),掛上去就能跑、也能對照手動下單。
- 想接 webhook 自動掛單 → 看 `breakout_v5_live.pine`。

**定案參數**:箱長 10 · RVOL 1.3 · EMA200 趨勢濾網 · 風險 5%/筆 · 槓桿上限 5x · 做空濾網「EMA200 下彎」· 保本 arm 3.0 ATR · 吊燈停利 3.5 ATR。

---

## 🧠 三個核心觀念(整個專案的精華)

1. **風險定額,不是固定槓桿**
   `倉位 = 風險額(當前權益的5%) ÷ 停損距離`。槓桿是「算出來的結果」、會隨波動自動浮動 → **根治爆倉**(實測風險賭到 25%/筆仍零爆倉)。

2. **牛市不做空(最大單一改進)**
   牛市裡的空單 9 筆 7 虧(誘空)。加上「EMA200 下彎才做空」→ 報酬 +202%→+334~500%、回撤 62%→18%、報酬/回撤 3.3→18+。**這是全專案最值錢的一步。**

3. **綁規則,不綁標的**
   標的會「變性」(BTC 可能機構化、0050 可能像台積電躺平)。解法不是選對標的,是**跑一籃子 + 定期重驗 + 熔斷退役**。見 `標的選擇.md`、`健康檢查.md`。

## 🎯 怎麼選標的(`標的選擇.md` 摘要)

策略 DNA = 收割「**高波動 × 強趨勢延續 × 肥尾**」。看到新標的,兩刀判:
1. 波動夠大嗎?(跑得出肥尾)
2. 動能還是均值回歸?(突破會續走、還是被打回?)

- ✅ **贏家**:幣的龍頭(BTC/ETH/SOL/BNB)、單一狂飆成長股(NVDA)、長多 ETF(0050)
- ❌ **輸家**:低波動大盤指數(S&P500、那指)、箱型權值股(台積電)

## 🩺 怎麼知道策略還活著(`健康檢查.md` 摘要)

**別用「年度 PF 比平均」(雜訊太大、會誤殺)。** 用這 4 個:
1. 目前回撤 vs 回測最大回撤(> ×1.3 → 🔴)
2. 距上次新高天數 vs 回測最長乾旱(> ×1.3 → 🔴)
3. 最近 25 筆滾動 PF(< 1.0 → 🔴)
4. 當前波動度百分位(< 20% → 🔴,領先示警「轉性」)

健康儀表板 Pine(顯示 🟢/🟡/🔴)見 `健康檢查.md`,可貼到任一策略末端。

## ⚠️ 三個永遠成立的誠實前提

1. **報酬靠少數幾筆怪物單** → 紀律是抱住大單、砍掉小單,接受多數是小虧。
2. **回測沒算永續資金費 + 滑點** → 實盤報酬會比回測再低一截。
3. **回測 ≠ 未來** → 標的會變性;靠「健康儀表板 + 定期重驗 + 熔斷」應對,不靠祈禱。
